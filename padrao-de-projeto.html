para defnirmos um padrao de fonte cores e ate espaçamentos tudo isso, temos um recurso que é criar variaveis para cor tamanho de fonte que eu gosto muito por sinal que é a classe root, é assim no css voce chama ela atraves dos dois pontos :root {} e tudo que estiver dentro dela podemos chamar no css atravez da função var(); no css

--nome-da-variavel:tipo-da-variavel
}
